# NFT Crypto VueX & TailwindCSS App - Rad Reindeers 6

A modern NFT marketplace application built with Vue.js, Vuex for state management, and TailwindCSS for styling. This project showcases a collection of NFT Rad Reindeers with various features and interactive components.

## Prerequisites

- Node.js (>= 12.x)
- npm or yarn

## Installation

1. Clone the repository:
```bash
git clone [repository-url]
cd vue-nft-rad-main
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm run serve
# or
yarn serve
```

The application will be available at `http://localhost:8080`

## Available Scripts

- `npm run serve` - Starts the development server
- `npm run build` - Builds the app for production
- `npm run lint` - Lints and fixes files

## Building for Production

To build the application for production, run:

```bash
npm run build
# or
yarn build
```

The built files will be generated in the `dist` directory.

## Project Structure

- `/src` - Source files
- `/public` - Static assets
- `/src/components` - Vue components
- `/src/store` - Vuex store modules
- `/src/assets` - Images and other assets
